<b>프로젝트 설명</b><br>
밤 하늘을 배경으로 별들을 이용하여 문자들을 구성할 수 있다.<br>
밤 하늘이라는 주제에 맞게 <b>별, 별자리, 별똥별, 달</b> 효과가 구현되어 있다.<br>
<b>마우스 왼쪽</b> 버튼을 누르고 있으면 별들이 마우스의 궤적을 따라서 그려진다.<br>
<b>마우스 오른쪽 버튼</b>을 누르면 미리 지정해둔 다음 문자들이 화면에 그려진다.<br>
